# VLMEdit: Text-Guided Image Editing Agent

VLMEdit — мультимодальный пайплайн для локального редактирования изображений по текстовой инструкции. Он находит нужный объект, строит его точную маску и изменяет только выбранную область, сохраняя остальную сцену.

## Задача

Цель проекта — реализовать controllable image editing. Пользователь задаёт команду, например: replace the umbrella with a yellow umbrella.

Система находит зонтик на изображении, строит маску и генерирует обновлённую версию объекта внутри этой области.

## Пайплайн

Input image + text instruction → Florence-2 phrase grounding → SAM mask generation → Canny edge extraction → ControlNet-guided Stable Diffusion Inpainting → Edited image → CLIP evaluation.

## Используемые модели

| Компонент | Роль |
|---|---|
| Florence-2 | Находит объект по текстовой фразе и возвращает bounding box. |
| SAM | Строит точную сегментационную маску объекта. |
| Canny | Выделяет контуры для сохранения структуры редактируемой области. |
| ControlNet | Добавляет структурный контроль к diffusion-генерации. |
| Stable Diffusion Inpainting | Генерирует новую версию объекта только внутри маски. |
| CLIP | Оценивает семантическое соответствие результата текстовой инструкции. |

## Возможности

- поиск объекта на изображении по тексту;
- построение маски объекта;
- локальная замена объекта через diffusion inpainting;
- использование Canny и ControlNet для сохранения формы и контуров;
- оценка результата через CLIP score.

## Запуск

Откройте ноутбук VLMEdit_Text_Guided_Image_Editing_Agent.ipynb в Google Colab, включите GPU и последовательно выполните ячейки.

## Пример

Инструкция: replace the umbrella with a yellow umbrella.

Ожидаемое поведение: пайплайн локализует зонтик, создаёт для него маску и заменяет его на жёлтый зонтик без изменения фона и остальных объектов сцены.

## Ограничения

- качество локализации зависит от формулировки запроса и видимости объекта;
- diffusion-модель может менять детали внутри маски сильнее, чем требуется;
- CLIP score — вспомогательная семантическая метрика, а не гарантия визуального качества.
